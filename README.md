\# Llama vs Ducky



This project hosts a simple image classification web application built using Google Teachable Machine and deployed via GitHub Pages.



\## What It Does



The page allows users to upload an image and classify it as either a Llama or a Ducky.



The model is loaded directly from Google Teachable Machine and runs entirely in the browser using TensorFlow.js.



There is \*\*no backend\*\*, all inference happens client-side.



---



\## How to Use



1\. Open the deployed page:



https://japsuu.github.io/uas-cloud-computing-module3/



2\. Click "Load Model"

3\. Upload an image (llama or duck)

4\. View prediction results and probabilities
